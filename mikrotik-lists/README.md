## Usage

### Simpliest Way

- Download **install‑cloudflare‑v4.rsc** and/or **install‑cloudflare‑v6.rsc** and upload them to your Mikrotik Router
- Run `/import file-name=install‑cloudflare‑v4.rsc` and/or `/import file-name=install‑cloudflare‑v6.rsc` from the Mikrotik terminal
- `cloudflare-download` and `dcloudflare-replace` will now be added to your Scripts
- `cf-dl` and `cf-rp` will be added to the Scheduler
